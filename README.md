# VelaBridge-openvela

## 项目名称

VelaBridge：基于 openvela 的多模态无障碍 AI 沟通与伴行终端

## 项目简介

VelaBridge 是一款基于 openvela 的多模态无障碍 AI 沟通与伴行终端，面向听障、视障和语言表达困难人群。设备通过实时字幕、语音播报、触感按键、震动提醒、盲人模式、一键求助和 AI 沟通摘要，实现“看得见、听得到、摸得着”的多模态无障碍交互体验。

本项目参加 openvela AI 硬件开发者大赛。第一阶段目标不是完成最终量产设备，而是在 3 天内完成一个可演示 MVP，用最小可行原型验证 openvela 端硬件交互、AI 文本处理和无障碍沟通流程。

## 目标用户

- 听障人群：需要将周围语音实时转换为可读字幕，并通过快捷回复降低沟通成本。
- 视障人群：需要通过语音菜单、触感按键和震动反馈完成无屏操作。
- 语言表达困难人群：需要用实体按键或 AI 推荐回复完成基础表达。
- 临时处于沟通困难场景的人群：如嘈杂环境、窗口服务、校园课堂、医院问诊和交通出行场景。

## 核心功能

- 实时字幕：将外界语音转换为文字显示。
- 快捷回复：通过实体按键触发常用语音播报。
- 盲人模式：通过语音菜单、触感按键和震动反馈完成无屏操作。
- 震动提醒：用不同震动模式提示状态和紧急信息。
- 一键求助：长按按键触发求助播报并记录日志。
- AI 摘要：根据沟通过程生成事件摘要。
- 隐私模式：默认不保存原始音频，仅保存用户确认后的文字摘要。

## 三天 MVP 目标

在 3 天内完成一个可演示的 VelaBridge MVP 原型，重点展示以下能力：

1. 屏幕显示字幕，验证听障用户的文字接收链路。
2. 实体按键触发快捷回复，验证低门槛表达链路。
3. 喇叭语音播报，验证对外沟通输出链路。
4. 震动提醒，验证非视觉、非听觉状态反馈链路。
5. 盲人模式，验证无屏语音菜单与触感交互。
6. 日志保存，验证设备事件记录。
7. AI 摘要生成，验证沟通过程的结构化总结。
