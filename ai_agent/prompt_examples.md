# VelaBridge 提示词示例

## 1. 字幕简化提示词

### 使用场景

听障用户需要在小屏幕上快速阅读外界语音内容时，将长句压缩为更短、更清晰的字幕。

### 输入示例

```text
请将下面这句话简化为适合小屏幕显示的短字幕，保留核心意思，字数尽量少：
“麻烦您能不能稍微说慢一点，因为我需要通过字幕理解您刚才说的内容。”
```

### 期望输出

```text
请说慢一点，我需要看字幕。
```

## 2. 快捷回复推荐提示词

### 使用场景

语言表达困难用户或听障用户在窗口服务、课堂、医院等场景中，需要快速选择一句合适回复并通过喇叭播报。

### 输入示例

```text
你是无障碍沟通设备的 AI 助手。请根据场景推荐一句简短、礼貌、可直接播报的回复。
场景：医院问诊，对方说得太快，用户没有听清。
```

### 期望输出

```text
我没有听清，请您说慢一点，或者写下来，谢谢。
```

## 3. 沟通摘要生成提示词

### 使用场景

用户希望在一次沟通结束后，保存经过确认的文字摘要，便于回顾，同时避免保存原始音频。

### 输入示例

```text
请根据以下设备事件日志生成一段简洁沟通摘要。不要添加日志中没有的信息。
1. 字幕：请到 3 号窗口办理。
2. 快捷回复：请说慢一点，谢谢。
3. 进入盲人模式：语音菜单已开启。
4. 紧急求助：我需要帮助。
```

### 期望输出

```text
本次沟通中，设备显示了“请到 3 号窗口办理”的字幕，用户触发快捷回复请求对方放慢语速，随后进入盲人模式，并触发了一次求助播报。
```
