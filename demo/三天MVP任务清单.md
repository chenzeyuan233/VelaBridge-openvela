# 三天 MVP 任务清单

## 三天总目标

完成一个可演示的 VelaBridge MVP 原型，展示以下功能：

1. 屏幕显示字幕
2. 按键快捷回复
3. 喇叭语音播报
4. 震动提醒
5. 盲人模式
6. 日志保存
7. AI 摘要生成

## 第一天任务

- 跑通 openvela 基础环境
- 完成屏幕显示测试
- 完成按键输入测试
- 完成震动马达测试
- 确认语音播报方案
- 创建 AI 文本处理脚本
- 保存 AI Coding 日志

## 第二天任务

- 完成字幕显示流程
- 完成快捷回复流程
- 完成盲人模式语音菜单
- 完成震动编码
- 完成日志保存

## 第三天任务

- 完整联调
- 录制 1–2 分钟演示视频
- 整理 README 和项目说明
- 准备下一阶段计划

## 验收标准

- openvela 端至少能模拟或真实触发屏幕、按键、震动和喇叭链路。
- AI Demo 脚本可在 Python 3 环境直接运行。
- 演示视频能够完整说明目标用户、核心功能和 MVP 完成度。
- 仓库文档、日志样例和 AI Coding 日志保持同步更新。
