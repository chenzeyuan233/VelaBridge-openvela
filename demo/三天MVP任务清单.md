# VelaBridge 三天 MVP 任务清单

## 三天目标

完成一个可演示的 VelaBridge MVP 原型，展示以下核心功能：

1. 屏幕显示字幕
2. 按键触发快捷回复
3. 喇叭语音播报
4. 震动提醒
5. 盲人模式
6. 日志保存
7. AI 摘要生成

## 第一天任务

- 跑通 openvela 基础环境
- 完成屏幕显示测试
- 完成按键输入测试
- 完成震动马达测试
- 完成语音播报方案确认
- 创建 AI 文本处理脚本
- 保存 AI Coding 日志

## 第二天任务

- 完成字幕显示流程
- 完成快捷回复流程
- 完成盲人模式语音菜单
- 完成震动编码
- 完成日志保存

## 第三天任务

- 整理 README 和项目说明
